# Collaboration Approach

The Flash-Guard project followed a collaborative Quality Engineering workflow where work was divided across Web UI, API, Database, and Mobile testing layers. GitHub pull requests and peer reviews were used before changes were merged into the main branch to ensure code quality, traceability, and team accountability. Team members worked together on documentation, evidence collection, defect management, and final project presentation.
