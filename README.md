# 📌 Project Overview

**Flash-Guard** is a production-grade fintech automation framework engineered to validate enterprise-level transactional systems across multiple testing layers.

This capstone project simulates a real-world financial ecosystem where reliability, compliance, security, and transactional integrity are mission-critical.

The framework integrates:

* 🌐 **Web UI Automation** using Playwright
* 🔌 **API Testing** using Postman & Newman
* 🗄️ **Database Validation** using SQL
* 📱 **Mobile Automation** using Appium
* ⚙️ **CI/CD Orchestration** using Github Actions
* 🐳 **Containerization** using Docker

The system validates:

✔ Secure authentication flows
✔ Financial transaction processing
✔ Ledger reconciliation
✔ Compliance & audit enforcement
✔ Cross-platform consistency
✔ End-to-End transfer lifecycle validation

# 🚀 Installation & Setup

## Clone Repository

```bash
git clone https://github.com/your-repo/flash-guard.git
```

## Install Dependencies

```bash
npm install
```

## Execute Playwright Tests

```bash
npx playwright test
```

## Execute Postman Collections

```bash
newman run collection.json
```

## Run Docker Environment

```bash
docker-compose up
```
